userMgtCtrl (controller in module kt.admin.userMgt )

Description

사용자관리 컨트롤러

Dependencies

  • $http

Methods

  • changePage(page, rowNum)

    페이지 전환을 한다.

    Parameters

    • page(optional) – {string=} –

      페이지 번호

    • rowNum(optional) – {string=} –

      행 갯수

  • deleteUser(userId)

    그리드에서 선택한 사용자를 삭제한다.

    Parameters

    • userId(optional) – {string=} –

      삭제할 사용자 ID

  • insertUser(data)

    새로운 사용자를 추가한다.

    Parameters

    • data(optional) – {object=} –

      추가할 사용자 폼 데이터

  • searchUser(searchInfo)

    사용자를 검색한다.

    Parameters

    • searchInfo(optional) – {string=} –

      검색 조건

  • selectUser(selectedUser)

    그리드에서 row를 클릭시 호출되어 사용자를 선택한다.

    Parameters

    • selectedUser(optional) – {object=} –

      그리드에서 선택한 row의 data

  • updateUser(data)

    사용자 정보를 업데이트 한다.

    Parameters

    • data(optional) – {object=} –

      추가할 사용자 폼 데이터

Properties

  • roleList

    [array] 역할 목록이다. 검색조건에서 사용된다.

  • searchInfo

    {object} 검색조건이다. 내부 속성으로 "userId" - 사용자아이디, "userNm" - 사용자이름, "svcId" - 서비스아이디, "roleId" - 권한아이디 들이 있다.

  • serviceList

    [array] 서비스 목록이다. 검색조건에서 사용된다.

  • userList

    {object} 사용자 목록이다. rows 프라퍼티에 list로 각 사용자 객체가 담겨있다.
